## 📌 About the Project

Eco-Mart is a modern e-commerce homepage built as a frontend assignment using React and Vite. The project is based on the provided Figma design and focuses on creating a responsive and reusable UI with clean component-based architecture.

## 🚀 Tech Stack

- React.js
- Vite
- CSS3
- React Icons
- Bootstrap Icons
